# 布局配置字段定义说明

本文档汇总了当前系统中定义的柜体、面板及元件的相关配置字段及其可选项，旨在帮助产品及技术团队了解目前的配置能力。

## 1. 柜体字段 (Cabinet Fields)

用于定义配电柜整体属性的字段。

| 标签 (Label) | 字段名 (Key) | 类型 (Type) | 选项 / 默认值 | 是否可编辑 |
| :--- | :--- | :--- | :--- | :--- |
| 柜体名称 | `cabinet_name` | 文本 (text) | - | 是 |
| 箱体分类 | `box_classify` | 选择 (select) | 配电箱 (默认), 户箱, 电表箱 | 是 |
| 箱体系列 | `series` | 选择 (select) | XM1 (默认), XM2, MZ, HW, DNB | 是 |
| 进线方式 | `inline_mode` | 选择 (select) | 上置 (默认), 左置 | 是 |
| 安装方式 | `install_type` | 选择 (select) | 户内暗装 (默认), 户内挂墙, 户内落地, 户外挂墙, 户外落地 | 是 |
| 固定方式 | `fixup_type` | 选择 (select) | 板式安装 (默认), 梁式安装 | 是 |
| 门型 | `door_type` | 选择 (select) | 左开门 (默认), 右开门, 双开门 | 是 |
| 进出线方式 | `cable_in_out_type` | 选择 (select) | 上进下出 (默认), 上进上出-左侧出线, 上进上出-右侧出线, 下进下出-左侧进线, 下进下出-右侧进线, 下进上出-左进右出, 下进上出-右进左出 | 是 |
| 宽度(W) | `cabinet_width` | 数字 (number) | 默认: 500 | 是 |
| 高度(H) | `cabinet_height` | 数字 (number) | 默认: 800 | 是 |
| 深度(D) | `cabinet_depth` | 数字 (number) | 默认: 200 | 是 |

## 2. 面板字段 (Panel Fields)

用于定义柜体内部面板属性的字段。

| 标签 (Label) | 字段名 (Key) | 类型 (Type) | 选项 / 默认值 | 是否可编辑 |
| :--- | :--- | :--- | :--- | :--- |
| 面板类型 | `panel_type` | 选择 (select) | 安装板 (默认), 门板 | 是 |
| 宽度(W) | `panel_width` | 数字 (number) | 默认: 500 | 是 |
| 高度(H) | `panel_height` | 数字 (number) | 默认: 800 | 是 |

## 3. 元件字段 (Part Fields)

用于定义具体电气元件（如断路器等）属性的字段。

| 标签 (Label) | 字段名 (Key) | 类型 (Type) | 选项 / 默认值 | 是否可编辑 |
| :--- | :--- | :--- | :--- | :--- |
| 进出线 | `in_line` | 布尔 (boolean) | - | 是 |
| 元件类型 | `part_type` | 文本 (text) | - | 是 |
| 元件型号 | `part_model` | 文本 (text) | - | 是 |
| 极数 | `pole` | 文本 (text) | - | 是 |
| 额定电流 | `current` | 文本 (text) | - | 是 |
| 宽(w) | `part_width` | 数字 (number) | 默认: 60 | 是 |
| 高(h) | `part_height` | 数字 (number) | 默认: 80 | 是 |
